Team/Player finding for competitive TF2
=======================================

Overview
--------

A database and browsing interface for LFT/LFP advertisements in the competitive TF2 community.

Supported Leauges:
- UGC Highlander
- ESEA 6v6
- ETF2l 6v6

Currently Supported Regions:
- North America
- Europe


Dates
-----

Project began on Sunday, August 10th, 2014
